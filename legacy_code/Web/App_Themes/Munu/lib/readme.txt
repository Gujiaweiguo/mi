File Description
1. nlsmenu.js. Main Nlsmenu javascript file
2. nlsmenu._nlstree_intg.js. NLSMenu Integration to NLSTree javascript file
3. nlsmenueffect.js. NLSMenu effect javascript file

6. nlsmenuext_xml.js. NlsMenu Extension for xml.
7. nlsmenuext_htm.js. NlsMenu Extension for loading from HTML.
