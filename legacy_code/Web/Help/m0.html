系统界面介绍
　　此文档是为MI.net系统提供的简明说明书。此说明书是对系统的功能模块和工作流进行简要说明。系统将功能模块与工作流做了严格的区分。如下图（图1-1）：
图(1-1)
  功能菜单区：根据当前操作用户的角色功能不同而有所不同。
  工作流区：当前操作用户角色所涉及到的工作流程。
  提示信息区：系统操作提示信息的显示区域。
  当前窗体刷新：刷新当前窗体到初始状态。
  系统管理：根据菜单，提供不同的管理功能。
　　浏览工具介绍
　　MI.net系统是基于.net平台的BS结构系统，本系统现仅支持基于IE内核的浏览器访问，最佳分辨率1024*768； 使用其他内核浏览器，我们不能保证系统界面的完全正常。
